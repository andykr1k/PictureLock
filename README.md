# PictureLock

## Frontend

### Setting up Enviroment
```
cd Frontend
cd Web
cd Picturelock
npm install
```

### Running Program
```
cd Frontend
cd Web
cd Picturelock
npm run dev
```
## Backend

### Setting up Enviroment
```
cd Backend
python -m venv env
pip install -r requirements.txt
```
### Running Program
```
cd Backend
cd src
flask run
```
